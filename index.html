A collaborative code planner and editor.

Collaborative code editing
because united we stand, divided we fall.

edit.ol is a realtime code planner and editor with chat support. It is used to collaborate with the peers to edit code in realtime. Multiple users can work on the same code file in real time.

edit.ol consists of two parts, discuss and develop. The discussion panel helps the developers to conceptualize and prioritize the tasks they want to do. The development panel has the code editor with chat interface.

edit.ol integrates smoothly with the existing authentication systems enabling users to log-in with their Facebook, Twitter and GitHub accounts.

edit.ol is built with Bootstrap, AngularJS on the front end and uses Firebase on the backend.

edit.ol is opensource!

The project is hosted on github. Please file an issue if you find any bugs. Thanks for your co-operation.
